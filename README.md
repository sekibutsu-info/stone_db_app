# みんなで石仏調査
[みんなで石仏調査](https://map.sekibutsu.info)のソースコードをas isで公開します。

## 開発・動作環境
開発当初は PHP 8.02／Laravel Framewok 9.19／Laravel Breeze 1.18でしたが、現在はPHP 8.2／Laravel Framework 11.0／Laravel Breeze 2.0で動作しています。
